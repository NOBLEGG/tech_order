# Tech Order

이 프로젝트는 `daily`, `weekly`, `monthly` 같은 인터벌에 따라
해야 할 일을 세팅하고, 이를 실행했는지 남기는 매니지먼트 앱이다.

이 프로젝트는 U.S. Air Force에서 쓰는
Technical Order라는 매뉴얼에서 힌트를 얻었다.
정비 플랜에 따라 무엇을 언제 해야 하는지 알 수 있는 짜임,
이에 따라 탱크나 레이더 같은 디바이스를 매니지먼트하는 것을
사람에게도 쓸 수 있겠다는 생각에서 시작했다.

여기서 사람은 하나의 디바이스처럼 다뤄진다.
만들어진 플랜을 따라가고 있는지 한눈에 볼 수 있는 앱이 되어야 한다.

## Core Concepts

- `Object`: 매니지먼트하고자 하는 것
- `Schedule`: 오브젝트에 붙는 반복 태스크
- `Completion`: 스케줄에 따라 행동을 어떻게 했는지

## 이 프로젝트의 포커스

- 어떤 일을 `언제` 해야 하는지 세팅한다
- 인터벌에 따른 플랜을 빠르게 만들고 업데이트한다
- 완료 메모를 남겨 매니지먼트를 잘하고 있는지 본다
